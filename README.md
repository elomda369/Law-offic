# موقع مكتب الأستاذ إسلام الدكروري للمحاماة

هذا المستودع يحتوي على ملفات الموقع الرسمي لمكتب الأستاذ إسلام الدكروري للمحاماة.

الموقع يحتوي على:
- الصفحة الرئيسية
- عن المكتب
- الخدمات القانونية
- مقالات قانونية
- نموذج طلب استشارة
- زر واتساب مباشر للتواصل

تم تصميم الموقع ليكون متوافق مع الموبايل وسهل الرفع على GitHub Pages.
